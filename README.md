## Quiz App
A simple and interactive quiz app that allows users to test their knowledge on various topics.

You can play around with it [here](https://maslowskilukasz.github.io/react-quiz/).

## Techonology used
- React
- TypeScript
- Vite
- Mantine
- Tanstack Query
- Phosphor Icons

## How to run
```
npm install
npm run dev
```
